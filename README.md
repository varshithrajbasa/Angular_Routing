# Angular Routing

This project includes Angular routing and nested(child) routing  

## Code

    {
    	path: 'route',
    	component: Component1,
    	children: [{
    			path: 'child1',
    			component: Component2,
    		},
    		{
    			path: 'child2',
    			component: Component3,
    		}
    	]
    }
